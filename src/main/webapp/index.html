MoSKito Counter

MoSKito Counter is a small application that allows us to track the usage of MoSKito Tools all over the world. Its sole purpose is to give us, the developers, the positive feedback of seeing people actually using the framework.

MoSKito Counter gathers no personal or other information, its only aim is to count the requests to MoSKito-Inspect and MoSKito-Control. You can see the worldwide requests to MoSKito here:

  • by Tools
  • inside MoSKito-WebUI
  • Accumulators: tool usage

You can switch the Counter off with a configuration option:
For MoSKito Essential: add file mskwebui.json to your classpath with following line:
{ "trackUsage": false }

For MoSKito Control: add following line to your moskitocontrol.json:
"trackUsage": false
